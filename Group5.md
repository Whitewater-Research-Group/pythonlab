# Group 5 Pythonlab project


## Description 
This application uses basic Python code to welcome new users to Pythonlab.




**setting up project from  github**
* we cloned the project from github using;
>git clone https://github.com/Whitewater-Research-Group/pythonlab.git

**opening the cloned repository**
* we then entered the repositry from our terminal using;
>cd pythonlab

**creating branch**
* we then created a new branch using;
> git checkout -b Group5

**making changes**
* we then edited the code in the main.py file and the markdown file;
 
**adding changes to the branch**
* we then added all our changes to the branch using:
> git add.

**commiting changes to the branch**
* we then commited the changes using;
> git commit -m "commit message"

**pushing the branch**
* then lastly we pushed our branch to the repository using;
> git push -u origin Group5
